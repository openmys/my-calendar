**TanStack Table의 아키텍처 패턴을 따른 헤드리스 캘린더 라이브러리**

## 🎯 핵심 특징

### TanStack Table 패턴 적용

- **헤드리스 코어**: UI와 로직의 완전한 분리
- **피처 시스템**: 모듈화된 기능 확장
- **타입 안전성**: TypeScript로 완전한 타입 추론
- **메모이제이션**: 성능 최적화된 계산

### 기본 사용법

```javascript
import {
  useReactCalendar,
  DateSelection,
  RangeSelection,
} from '@openmys/my-calendar';

function MyCalendar() {
  const calendar = useReactCalendar({
    features: [DateSelection, RangeSelection],
    currentDate: new Date(),
  });

  const month = calendar.getMonth();
  const weeks = month.getWeeks();

  return (
    <div>
      {weeks.map(week => (
        <div key={week.id} className='week'>
          {week.getDays().map(day => (
            <div
              key={day.id}
              onClick={() => day.toggleSelected?.()}
              style={{
                backgroundColor: day.getIsSelected?.() ? 'blue' : 'white',
              }}
            >
              {day.getValue().getDate()}
            </div>
          ))}
        </div>
      ))}
    </div>
  );
}
```
